## Organization Principles

- **Progressive disclosure:** Start light (README), get deeper as needed
- **Single source of truth:** Each topic lives in one primary doc with cross-links
- **Cross-linked:** Docs link to related docs, no duplication
- **Role-based:** Quick paths for developers, reviewers, AI agents, etc.
